Requirements/Wymagania
 1. php 7.2
 2. screen

 Intalation/Instalacja
 1. cd /home
 2. wget https://github.com/stalkerlifehack/PublicChannelMonitor/archive/master.zip
 3. unzip master.zip
 4. mv PublicChannelMonitor-master PublicChannelMonitor
 5. cd PublicChannelMonitor
 6. chmod 0775 run 
 7. cd inc/configs
 8. cp config.php.SAMPLE_EN config.php /  cp config.php.SAMPLE_PL config.php

Commands/Komendy
    ./run start
    ./run stop
    ./run restart
    ./run daemon
